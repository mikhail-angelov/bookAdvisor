# Book Advisor

Book Advisor is a Next.js 14 app for browsing audiobook torrents crawled from Rutracker.

- Frontend: Next.js App Router, React, Zustand
- Backend: Next.js route handlers
- Database: SQLite with Drizzle ORM
- Auth: JWT magic-link login
- Extra services: crawler for Rutracker data, optional Qdrant/Hugging Face for recommendations
